# 在线音乐播放器

一个功能完整的现代化在线音乐播放器，支持完整的播放控制和管理功能。

## 🎵 功能特性

### 播放控制界面
- ✅ 播放/暂停按钮
- ✅ 上一首/下一首按钮  
- ✅ 进度条（可拖拽）
- ✅ 音量控制滑块
- ✅ 随机播放切换
- ✅ 循环播放模式（单曲/列表/关闭）

### 歌曲信息显示
- ✅ 当前播放歌曲名称
- ✅ 艺术家信息
- ✅ 播放时长显示
- ✅ 专辑信息显示

### 播放列表管理
- ✅ 播放列表显示
- ✅ 当前播放歌曲高亮
- ✅ 点击列表中的歌曲可直接播放
- ✅ 实时播放状态指示

### 示例音乐资源
- ✅ 5首不同风格的示例音乐
- ✅ 涵盖电子、氛围、电影配乐、古典等风格
- ✅ 所有音乐文件可正常播放

### 界面设计
- ✅ 现代化、简洁的设计风格
- ✅ 深色主题为主
- ✅ 响应式设计，支持移动设备
- ✅ 流畅的动画效果
- ✅ 直观的用户交互

## 📝 使用说明

1. 打开音乐播放器网页
2. 从播放列表中选择要播放的歌曲
3. 使用播放控制按钮进行播放管理
4. 拖拽进度条跳转到指定位置
5. 调节音量滑块控制音量大小
6. 切换播放模式（随机、循环等）

## 🔧 本地开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build
```
